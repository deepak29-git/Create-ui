Avatar

Avatars are usually used in the user profile. Create ui provides different shapes and sizes of avatar.

Profile Avatar

Profile Avatar can be creted using "avatar" class inside the image tag.

Sizes Avatar

Create Ui provides different size of avatar sm | md | lg with existing avatar class.

Squre Avatar

We can create the square shape avatar by using square class with existing avatar class in different sizes.
